# 🚀 Il mio primo viaggio con Flask

Questa è la mia prima applicazione web completa, sviluppata utilizzando il framework **Flask**. Il progetto è iniziato come un esperimento didattico in Italia ed è evoluto rapidamente in un'app funzionale.

## 🛠 Funzionalità implementate:
* **Backend in Python:** Gestione completa delle rotte e della logica lato server.
* **Moduli Interattivi:** L'app è in grado di ricevere input dall'utente e processarli in tempo reale.
* **Archiviazione Dati (V1.0):** Sistema di salvataggio su file di testo (`.txt`) — i miei obiettivi rimangono memorizzati anche dopo il riavvio del server.
* **UI Moderna:** Interfaccia pulita e reattiva realizzata con **Bootstrap 5**, con stili personalizzati e un design moderno.

## 📈 Roadmap (Prossimi passi):
* **Database SQL:** Passaggio dai file di testo a un database professionale (SQLite/PostgreSQL).
* **Docker:** Containerizzazione dell'app per avviarla su qualsiasi server con un solo click.
* **Autenticazione Utente:** Sistema di registrazione e aree personali protette.
* **Integrazione API:** Aggiunta di dati esterni come meteo o tassi di cambio.
